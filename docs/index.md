# Welcome to Vega

**Vega** is your personal AI agent. Installed locally. Private by default. Extensible by design.

## One Command

```bash
curl -fsSL https://getvega.sh | bash
```

## What makes Vega different?

| Other agents | Vega |
|-------------|------|
| Chat in a browser tab | Works in your terminal |
| Cloud-only data | Local-first, always |
| Forget everything between sessions | Context Tree remembers |
| No audit trail | Every action logged |
| Single model | Route any LLM |
| Closed source | MIT open source |
